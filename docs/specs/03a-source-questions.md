# Spec 03a — Source Questions (Authoritative)

## Purpose

The 100 original epistemic questions for the Construct's grid. These are the authoritative source from which Spec 03 derives point questions (construct node content) and spectrum questions (SPECTRUM_OPPOSITION edge content).

This document is not parameterized. These questions describe the universal structural/epistemic nature of each position and each spectrum, independent of any branch.
